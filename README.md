# AIPM v2 - Multi-Project Orchestrator

Autonomous project management for large mono-repos.

## Features
- **Auto-Discovery**: Scans tree for projects (Cargo.toml, package.json, etc.)
- **Grounded State**: Persistent tracking of git and test state
- **Parallel Agents**: Runs multiple 'pi' agents concurrently
- **Health Management**: Automated back-off for failing projects

## Usage
```bash
python3 main.py
```
